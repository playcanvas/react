#  Building Blocks for 3D

**A set of high-level 3D building blocks** — interactive viewers, spatial layouts, and controls—designed to help you compose immersive scenes with minimal setup.

## Splat Viewer <Badge variant="secondary" className="-mt-2">New</Badge>

A component for displaying Gaussian splats in the browser.

Includes built-in camera controls, loading states, and optional UI slots for overlays or actions.  
Supports large assets with lazy loading and automatic framing. Use it in modals, pages, or full-screen scenes.

### Installation

Install the component from your command line.

```bash copy
npx shadcn@latest add https://playcanvas-react.vercel.app/api/r/splat-viewer
```

### Anatomy

Import the component.

```jsx
import { SplatViewer } from "@components/ui/splat-viewer";
export default () => <SplatViewer />;

```
